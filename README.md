# 🌸 Iris Flower Classification

## 📌 Project Overview

This project is developed as part of the Horizon TechX Data Science Internship Program.

The main objective of this project is to classify Iris flower species using Machine Learning algorithms based on flower measurements.

---

## 🚀 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

## 📂 Dataset

The dataset used in this project is the Iris Flower Dataset.

Features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Target:

* Setosa
* Versicolor
* Virginica

---

## 🤖 Machine Learning Model

This project uses:

* Logistic Regression

---

## 📊 Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Preprocessing
4. Train-Test Split
5. Model Training
6. Prediction
7. Accuracy Evaluation
8. Data Visualization

---

## 📈 Output

The model predicts the species of Iris flowers with high accuracy.

---

## ▶️ How to Run

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Upload the dataset file (`iris.csv`).
3. Run all cells.
